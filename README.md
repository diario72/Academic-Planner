# Academic Planner

An academic planning application developed with Python and PostgreSQL that enables students to efficiently organize courses and manage structured semester schedules.

## Features
- Add courses
- View all courses
- View semester schedule
- Search courses
- Delete courses

## Technologies Used
- Python
- PostgreSQL

## How to Run
1. Create a PostgreSQL database
2. Update the database credentials in the code
3. Run:
   ```
   python academic_planner.py
   ```
