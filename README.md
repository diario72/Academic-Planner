# Student Course Planner

A simple academic planning application built with Python and PostgreSQL.

## Features
- Add courses
- View all courses
- View semester schedule
- Search courses
- Delete courses

## Technologies Used
- Python
- PostgreSQL
- Git

## How to Run
1. Create a PostgreSQL database
2. Update the database credentials in the code
3. Run:
   python academic_planner.py# Academic-Planner
